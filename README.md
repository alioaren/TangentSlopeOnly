# TangentSlopeOnly

A simple Java program that calculates the slope of the tangent line to a function at a given point.

## Description
This project demonstrates how to compute the slope of a tangent using basic calculus concepts.
It is implemented in Java and intended for learning and educational purposes.

## Features
- Calculates the slope of a tangent line
- Simple and clean Java implementation
- Beginner-friendly code

## Requirements
- Java JDK 8 or higher
